## Level grid

- 24 tiles
  - 6 rows
  - 4 columns
- margins
  - 4 * (20sw / 14) horizontal for grid
  - (sh - 6 * (tile size) - 12 * (tile margin)) / 2 - vertical for grid to center it

## Level tiles

-  19.5sw - tile size
- (20sw / 14) - around margins
- solid white if locked
- white with number if opened
- tiled if solved
